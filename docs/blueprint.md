# **App Name**: ShutterForge Portfolio

## Core Features:

- Dynamic Portfolio Showcase: Display a categorized, high-resolution photography portfolio with image lightboxes or detailed project pages.
- Interactive Booking Inquiry: A modal form allowing clients to submit contact information and project details, with client-side form validation.
- Photographer Biography: A dedicated 'About' section detailing the photographer's philosophy, experience, and artistic approach.
- Services Overview: Outline of photography services offered, including types of shoots (e.g., brand, portrait) and typical project scope.
- Direct Contact Links: Easy access to direct communication channels such as WhatsApp and social media profiles (Instagram, Behance, LinkedIn).
- AI Social Media Caption Tool: Leverage AI to generate unique and engaging social media captions or hashtag suggestions based on project details or themes, assisting with marketing.
- Fully Responsive Design: Ensure the application provides an optimal viewing and interaction experience across all device sizes (mobile, tablet, desktop).

## Style Guidelines:

- Color Palette based on Elegance and Modernity in a dark theme: Primary accenting color (e.g., for buttons/links): #4D4573 (a deep, rich blue-purple). Background color: #17141D (a very dark, subtly purplish gray). Highlight/Accent color: #66A8E5 (a brighter, cooler blue) to draw attention to interactive elements.
- Headline font: 'Cinzel' (serif) for an elegant and impactful title presentation. Body font: 'Inter' (sans-serif) for clean readability and modern professionalism.
- Utilize Lucide Icons for clean, consistent, and scalable vector icons throughout the interface, maintaining a minimalist and modern aesthetic.
- A clean, modern layout with ample white space (dark space in this theme) to emphasize imagery. Implement a responsive grid system for optimal content display on all devices.
- Subtle, smooth transition animations for elements on scroll (like portfolio items appearing) and UI interactions (such as modal open/close) to enhance user experience without distraction. A 'loading' animation should precede content display.
